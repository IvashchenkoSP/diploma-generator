# Diploma Generator

Spring Boot приложение для генерации дипломов с поддержкой кириллицы.

## Features
- Генерация дипломов из PDF шаблонов
- Поддержка кириллических шрифтов
- Предпросмотр в реальном времени
- Пакетная генерация

## Deployment
Deployed on Railway: https://diploma-generator.up.railway.app
